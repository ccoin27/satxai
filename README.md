# SatX AI — Discord чат-бот с нейросетью

Чат-бот для Discord — участник комьюнити хостинга. Локальная нейросеть (Ollama), память, досье, поиск.

## Возможности

- **Память канала** — помнит всю переписку в канале (до 80 сообщений), не только личный диалог
- **Личная память** — до 50 сообщений на пользователя для контекста
- **Автоанализ** — каждые 25 сообщений обновляет досье и резюме обстановки в фоне
- **Досье** — `!анализ` или автоматически: профили участников и обстановка в чате
- **Ведёт себя как участник** — отвечает при упоминании или при фразах «бот», «что думаешь», «твоё мнение» и т.п.
- **Зумерский стиль** — сленг, мемы, сложно отличить от живых
- **Пассивные гифки** — читает чат и иногда реагирует гифкой на сообщения (лол, рофл, кринж и т.д.)
- **Поиск** — по запросам «поищи», «найди», «кто такой», «что такое»

## Требования

- Python 3.9+
- [Ollama](https://ollama.ai) с установленной моделью
- Discord Bot Token

## Установка

```bash
pip install -r requirements.txt
```

Скопируй `.env.example` в `.env` и заполни `DISCORD_TOKEN`. Для гифок — `TENOR_API_KEY` ([получить](https://developers.google.com/tenor/guides/quickstart)).

**Создание кастомной модели** (hermes3:8b + промпт для чата хостинга):

```bash
ollama pull hermes3:8b
ollama create satx-host -f Modelfile
```

Или: `bash setup_model.sh`

## Запуск

```bash
python main.py
```

## Логи промптов

Промпты пишутся в `prompt_logs/` (папка из `PROMPT_LOG_DIR` в `.env`).

## Использование

- **@бот** — ответ с учётом всей переписки в канале и досье
- **«бот что думаешь»**, **«твоё мнение»**, **«а ты как»** — ответит без упоминания
- `!анализ` — ручной анализ чата (только для админов)
- `!архив` — сканирование и анализ канала архива (только для админов)

## Структура БД

**intelligence.db:**
- `logs` — личная история (user/assistant) на пользователя
- `channel_logs` — переписка в канале (общая память)
- `dossiers` — профили пользователей
- `channel_state` — резюме обстановки в канале (темы, тон)

**archive.db** (отдельная база):
- `raw_messages` — все сообщения с ссылками и вложениями (без дубликатов по message_id)
- Автосканирование канала 1371191476054393066 при старте
- `!архив` — ручной запуск сканирования и анализа
